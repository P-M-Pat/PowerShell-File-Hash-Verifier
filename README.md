# PowerShell-File-Hash-Verifier
A PowerShell script that uses a GUI file selector to calculate the selected file's SHA256 hashes and compares it with the checksum provided by user.

## ⚠️ Practice Project Disclaimer

This project was developed primarily as a **personal learning exercise** to practice PowerShell scripting and GUI integration. It is provided **as-is** and is **not guaranteed** to work properly everywhere.
